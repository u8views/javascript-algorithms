# Квадратний корінь (метод Ньютона)

_Прочитайте це іншими мовами:_
[_English_](README.md).

У числовому аналізі, галузі математики, існує кілька алгоритмів знаходження
квадратного кореня або методів обчислення головного квадратного кореня невід'ємного
дійсного числа. Оскільки, як правило, корені функції не можуть бути точно обчислені.
Алгоритми знаходження коренів надають наближення коренів у вигляді чисел з рухомою
крапкою.

Знаходження ![](https://wikimedia.org/api/rest_v1/media/math/render/svg/bff86975b0e7944720b3e635c53c22c032a7a6f1) є
еквівалентним вирішенню рівняння ![](https://wikimedia.org/api/rest_v1/media/math/render/svg/6cf57722151ef19ba1ca918d702b95c335e21cad)  для
додатнього `x`. Тому можна використовувати будь-який загальний числовий алгоритм знаходження коренів.


**Метод Ньютона** (також відомий як метод Ньютона-Рафсона), названий на честь
_Ісаака Ньютона_ та _Джозефа Рафсона_, є одним з прикладів алгоритмів знаходження коренів. Це
метод знаходження послідовно кращих наближень до коренів дійсної функції.

Почнемо з пояснення загальної ідеї методу Ньютона, а потім застосуємо його до нашого конкретного
випадку знаходження квадратного кореня числа.

## Загальна ідея методу Ньютона-Рафсона

Загальна ідея методу Ньютона-Рафсона в одній змінній така:

Метод починається з функції `f`, визначеної на дійсних числах `x`, похідної функції `f'` та початкової 
догадки `x0` для кореня функції `f`. Якщо функція задовольняє передумови, встановлені при похідної формули, 
та початкова догадка достатньо близька, то краща наближена величина `x1` визначається як:

![](https://wikimedia.org/api/rest_v1/media/math/render/svg/52c50eca0b7c4d64ef2fdca678665b73e944cb84)

Геометрично точка `(x1, 0)` є перетином вісі `x` та дотичної до графіка функції `f` у точці `(x0, f (x0))`.

Процес повторюється за формулою:

![](https://wikimedia.org/api/rest_v1/media/math/render/svg/710c11b9ec4568d1cfff49b7c7d41e0a7829a736)

доки не буде досягнуто достатньо точного значення.

![](https://upload.wikimedia.org/wikipedia/commons/e/e0/NewtonIteration_Ani.gif)

## Метод Ньютона у знаходженні квадратного кореня

Як вже було згадано вище, знаходження 
![](https://wikimedia.org/api/rest_v1/media/math/render/svg/bff86975b0e7944720b3e635c53c22c032a7a6f1) 
є еквівалентним розв'язуванню рівняння 
![](https://wikimedia.org/api/rest_v1/media/math/render/svg/6cf57722151ef19ba1ca918d702b95c335e21cad) 
для позитивного `x`.


Похідна функції `f(x)` у випадку задачі знаходження квадратного кореня дорівнює `2x`.

Після застосування формули Ньютона (див. вище) ми отримуємо наступне рівняння для ітерацій нашого алгоритму:

```text
x := x - (x² - S) / (2x)
```

Вираз `x² - S` показує, наскільки віддалено від `x²` потрібний нам результат, а ділення на `2x` є похідною `x²`, 
щоб масштабувати те, наскільки ми змінюємо `x` залежно від того, наскільки швидко змінюється `x²`.
